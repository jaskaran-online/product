CSS Box Model Tutorial

What is the CSS Box Model?

The CSS box model describes how elements are rendered on a webpage. Every HTML element is essentially a rectangular box consisting of four main areas:

CONTENT
  1. Content: The actual content of the element (text, images, etc.)
  2. Padding: Space between content and border
  3. Border: A line around the padding and content
  4. Margin: Space outside the border

Box-Sizing Property

The box-sizing property controls how the total width and height of an element is calculated:

content-box (default)

200px × 100px

Total width: 200px + 40px padding + 10px border + 40px margin = 290px

.box { width: 200px; height: 100px; padding: 20px; border: 5px solid; margin: 20px; box-sizing: content-box; }

border-box

200px × 100px

Total width: 200px (includes padding + border) + 40px margin = 240px

.box { width: 200px; height: 100px; padding: 20px; border: 5px solid; margin: 20px; box-sizing: border-box; }

Margin Examples

Margins create space outside the element's border:

margin: 50px 20px 30px 40px;
(top right bottom left)
/* Individual margins */ margin-top: 50px; margin-right: 20px; margin-bottom: 30px; margin-left: 40px; /* Shorthand */ margin: 50px 20px 30px 40px; /* top right bottom left */ margin: 20px 40px; /* top/bottom left/right */ margin: 20px; /* all sides */

Padding Examples

Padding creates space inside the element, between content and border:

padding: 40px 20px 30px 50px;
(top right bottom left)
/* Individual padding */ padding-top: 40px; padding-right: 20px; padding-bottom: 30px; padding-left: 50px; /* Shorthand */ padding: 40px 20px 30px 50px; /* top right bottom left */ padding: 20px 40px; /* top/bottom left/right */ padding: 20px; /* all sides */

Border Examples

Borders surround the padding and content:

Solid Border
border: 5px solid #e74c3c;
Dashed Border
border: 5px dashed #3498db;
Dotted Border
border: 5px dotted #2ecc71;
Double Border
border: 8px double #9b59b6;
/* Border shorthand */ border: width style color; border: 5px solid #e74c3c; /* Individual border properties */ border-width: 5px; border-style: solid; border-color: #e74c3c; /* Individual sides */ border-top: 2px dashed blue; border-right: 4px solid red; border-bottom: 1px dotted green; border-left: 3px double purple;

Box Model Calculations

Property content-box border-box Description
width/height Content only Content + Padding + Border What the width/height property controls
Total Width width + padding + border + margin width + margin Space occupied on page
Content Area width × height width - padding - border Available space for content

Key Takeaways

  • content-box (default): width/height applies to content only
  • border-box: width/height includes padding and border
  • Margin doesn't affect element's size, only spacing
  • Padding increases element's total size (unless border-box)
  • Border is part of element's visual size
  • Use box-sizing: border-box for easier sizing calculations
/* Recommended global reset */ * { box-sizing: border-box; } /* This makes all elements use border-box sizing */
