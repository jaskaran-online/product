1. üìã Table Basics

Basic Table Structure

<table> <thead> <tr> <th>Header 1</th> <th>Header 2</th> <th>Header 3</th> </tr> </thead> <tbody> <tr> <td>Data 1</td> <td>Data 2</td> <td>Data 3</td> </tr> </tbody> </table>

Example Output:

Product Price Stock
Laptop $999 15
Mouse $25 50
Keyboard $75 30
üí° Key Elements:
  • <table> - Main container
  • <thead> - Table header section
  • <tbody> - Table body section
  • <tr> - Table row
  • <th> - Table header cell
  • <td> - Table data cell

Complete Attribute Reference

Attribute Element Description Values Example
border table Sets table border width pixel value border="1"
cellpadding table Space inside cells pixel value cellpadding="10"
cellspacing table Space between cells pixel value cellspacing="5"
width table, td, th Sets element width pixels or % width="100%"
height table, td, th Sets element height pixels or % height="200px"
colspan td, th Spans multiple columns number colspan="2"
rowspan td, th Spans multiple rows number rowspan="3"
align table, tr, td, th Horizontal alignment left, center, right align="center"
valign tr, td, th Vertical alignment top, middle, bottom valign="middle"
bgcolor table, tr, td, th Background color color name or hex bgcolor="#ff0000"
scope th Defines header scope col, row, colgroup, rowgroup scope="col"
headers td, th Associates with header header id headers="h1"
abbr th, td Abbreviation for cell text abbr="Population"
axis td, th Categorizes cells text axis="financial"

Attribute Examples in Action

1. Colspan and Rowspan Example

<table border="1" cellpadding="10"> <tr> <th colspan="2">Sales Report</th> <th rowspan="2">Total</th> </tr> <tr> <th>Q1</th> <th>Q2</th> </tr> <tr> <td>$1000</td> <td>$1500</td> <td>$2500</td> </tr> </table>
Sales Report Total
Q1 Q2
$1000 $1500 $2500

Column Groups (colgroup)

<table border="1"> <colgroup> <col style="background-color: #f0f0f0;"> <col span="2" style="background-color: #e8f4f8;"> </colgroup> <tr> <th>Name</th> <th>Age</th> <th>City</th> </tr> <tr> <td>John</td> <td>25</td> <td>NYC</td> </tr> </table>
Name Age City
John 25 NYC

Accessibility Features

<table> <caption>Monthly Sales Data</caption> <thead> <tr> <th id="month" scope="col">Month</th> <th id="sales" scope="col">Sales</th> </tr> </thead> <tbody> <tr> <td headers="month">January</td> <td headers="sales">$5000</td> </tr> </tbody> </table>

4. üé® Styling Examples

CSS Styling Methods

/* CSS Styles */ .styled-table { border-collapse: collapse; margin: 25px 0; font-size: 0.9em; font-family: sans-serif; min-width: 400px; box-shadow: 0 0 20px rgba(0, 0, 0, 0.15); } .styled-table thead tr { background-color: #009879; color: #ffffff; text-align: left; } .styled-table th, .styled-table td { padding: 12px 15px; } .styled-table tbody tr { border-bottom: 1px solid #dddddd; } .styled-table tbody tr:nth-of-type(even) { background-color: #f3f3f3; } .styled-table tbody tr:last-of-type { border-bottom: 2px solid #009879; } .styled-table tbody tr.active-row { font-weight: bold; color: #009879; }
